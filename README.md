# type-typer-yet-typer
《Type Typer Yet Typer》是一本关于类型系统的小册子。
在这本小册子中，笔者将尽可能地使用简单易懂的方式介绍编程语言类型系统的二三事，让大家对类型系统有一个基本的认识。

《Type Typer Yet Typer》的名字neta自游戏《Undertale》中W.D.Gaster的同人曲《Dark Darker Yet Darker》。

## 编译
